Tool: GaudiViewX (Insilichem)

The GaudiViewX tool is a graphic interface for exploring, visualization, and analysis for the GaudiMM suite solutions.

The tool displays a field with all the Gaudi solutions. When the user click one or more of the solutions, the a type-in field. When the user hits the Return key, any text in the field will be shown in the log as its HTML equivalent. That means that the text: 

	<a href="http://www.cgl.ucsf.edu/chimerax">ChimeraX home page</a>
will be logged as:

ChimeraX home page

Choosing Clear from the tool context menu will clear the text field

Insilichem, UAB, Barcelona, Spain / May 2019
